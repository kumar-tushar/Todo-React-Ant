# Todo App (React & Ant)


## Quick Start

clone this repo:

```sh
$ git clone https://github.com/kumar-tushar/Todo-React-Ant.git
```

Move to the project dir:

```sh
$ cd Todo-React-Ant
```

With Docker just run:

```sh
$ docker-compose up -d
```
